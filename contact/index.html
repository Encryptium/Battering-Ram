Report Suspicious Activity

Please note that we use third-party ads. It is difficult to verify if every ad served on Battering Ram is trustworthy. If any advertisement displayed or served on the website seems suspicious or leads to a manipulative website, please report this activity via the Feedback Form located at the bottom right immediately.

Suspicious activity is classified as the following:


  • Ads containing malware
  • Ads covering the entire website
  • Ads with inappropriate content
  • Games containing malware
  • Unexpected website/game behavior
  • Website bugs
